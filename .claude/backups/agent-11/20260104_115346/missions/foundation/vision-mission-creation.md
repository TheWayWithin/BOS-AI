# Mission: Vision & Mission Creation

**Mission Type:** Foundation Document Creation
**Duration:** 45-90 minutes
**Output:** `/documents/foundation/vision-mission.md`

---

## Mission Objective

Create a compelling Vision & Mission document that establishes the strategic North Star for the business, guiding all future decisions and operations.

## Required Agents

- `strategic-opportunity` (primary) - Strategic positioning and long-term thinking
- `brand-strategy` (support) - Messaging clarity and emotional resonance
- `market-intelligence` (support) - Market context and competitive landscape

## PHASE 0: MODE SELECTION

**Before proceeding, ask the user:**

> **How would you like to create your Vision & Mission document?**
>
> | Mode | Description | Time |
> |------|-------------|------|
> | **Auto** | I'll create the complete document based on your inputs. You review and refine at the end. | 15-20 min |
> | **Engaged** | We work section by section. I draft, present different perspectives, you approve before moving on. | 45-60 min |
>
> Which mode works for you today?

**Wait for user response before proceeding.**

---

## PHASE 1: CONTEXT GATHERING

Regardless of mode, first gather context:

1. **Read provided context files** (PRD, ideation docs, market research)
2. **Ask clarifying questions if needed:**
   - What business/product is this for?
   - What's the timeframe for the vision (5 years? 10 years?)
   - Any existing mission/vision to build on?
   - Key differentiators or values important to founder?

## AUTO MODE EXECUTION

If user selected **Auto**:

### Step 1: Generate Complete Document

Create the full Vision & Mission document with these sections:

```markdown
# Vision & Mission

## Vision Statement
[10-year aspirational destination - what the world looks like when you succeed]

## Mission Statement
[How you'll achieve the vision - your unique approach and value delivery]

## Core Values
1. [Value 1] - [Why it matters]
2. [Value 2] - [Why it matters]
3. [Value 3] - [Why it matters]
[3-5 values that guide decisions]

## Strategic Pillars
1. [Pillar 1] - [Description]
2. [Pillar 2] - [Description]
3. [Pillar 3] - [Description]
[3-4 foundational strategic themes]

## Hedgehog Concept
- **What we can be best at:** [Unique capability]
- **What drives our economic engine:** [Revenue/value mechanism]
- **What we're passionate about:** [Core motivation]
- **The Intersection:** [Strategic focus area]

## North Star Metric
[Single metric that captures value delivery to customers]
```

### Step 2: Present for Review

> Here's your complete Vision & Mission document.
>
> [Show full document]
>
> **What would you like to refine?**
> - Vision statement
> - Mission statement
> - Core values
> - Strategic pillars
> - Hedgehog concept
> - North Star metric
> - Or approve as-is

### Step 3: Iterate Until Approved

Make requested changes and re-present until user approves.

### Step 4: Save Document

Save to `/documents/foundation/vision-mission.md`
