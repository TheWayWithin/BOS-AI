# Mission: Brand Style Guide Creation

**Mission Type:** Foundation Document Creation
**Duration:** 45-90 minutes
**Output:** `/documents/foundation/brand-style-guide.md`

---

## Mission Objective

Create a comprehensive Brand Style Guide that establishes visual identity, brand voice, and usage guidelines to ensure consistent brand presentation across all touchpoints.

## Required Agents

- `brand-strategy` (primary) - Brand development and identity
- `content-creation` (support) - Voice and messaging
- `campaign-execution` (support) - Practical application

## PHASE 0: MODE SELECTION

**Before proceeding, ask the user:**

> **How would you like to create your Brand Style Guide?**
>
> | Mode | Description | Time |
> |------|-------------|------|
> | **Auto** | I'll create the complete brand guide based on your inputs. You review and refine at the end. | 20-30 min |
> | **Engaged** | We work section by section. I draft, present perspectives, you approve before moving on. | 45-60 min |
>
> Which mode works for you today?

**Wait for user response before proceeding.**

---

## PHASE 1: CONTEXT GATHERING

Gather essential context:

1. **Read provided context files** (vision-mission, positioning, client blueprint)
2. **Ask clarifying questions:**
   - Do you have existing brand elements (logo, colors)?
   - What brands do you admire? Why?
   - What personality should your brand have?
   - Any domains secured? (for digital presence)
